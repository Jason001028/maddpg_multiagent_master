# 项目框架说明（MADDPG 多智能体协同搜索）

本文档对当前项目进行**文件树展示**与**逐模块介绍**，方便快速理解代码组织、训练流程与实验产物。

---

## 1. 项目概览

该项目是一个基于 **MADDPG** 的多智能体网格世界协同搜索/导航实现，采用多进程结构：

- **Actor**：与环境交互采样
- **Learner**：从回放池采样并更新网络
- **Evaluator**：固定间隔评估并输出可视化结果

训练入口为 `train.py`，参数集中在 `arguments.py`。

## 3. 根目录文件逐个介绍

### `train.py`
- 项目训练主入口。
- 负责初始化环境参数、模型、进程/队列，并启动训练流程。

### `arguments.py`
- 训练与环境相关超参数配置文件。
- 一般包括：学习率、batch size、训练轮次、随机种子、评估间隔等。

### `rollout_test.py`
- 使用已保存策略做 rollout（回放测试）。
- 用于观察训练后策略在环境中的行为表现。

### `plot.py`
- 对训练日志（如 `training_data.csv`）进行绘图。
- 输出 reward/成功率等曲线图。

### `collection_experiments.py`
- 用于实验采样/可视化的辅助脚本（偏实验性质）。

### `origin_obstacle_states.txt`
- 固定障碍布局数据。
- 用于保证不同实验间环境一致性，提升可复现性。

### `README.md`
- 项目简要说明、目录概览与示意图展示。

### `agent_cover.xlsx`
- 与实验记录或覆盖统计相关的数据文件（非核心训练代码）。

## 4. `core/` 训练核心模块逐个介绍

### `core/model.py`
- 定义神经网络结构：
  - `actor`：策略网络
  - `critic`：价值网络
  - `Net`：模型容器/管理

### `core/actor.py`
- 采样进程工作函数 `actor_worker`。
- 使用当前策略与环境交互，收集轨迹数据。

### `core/buffer.py`
- 回放池实现（`replay_buffer`）。
- 负责 `push/sample`，为离线小批量训练提供数据。

### `core/learner.py`
- 学习进程主逻辑：
  - 从 buffer 取样
  - 计算损失
  - 更新 actor/critic
  - 软更新 target network

### `core/evaluator.py`
- 评估进程逻辑：定期评估当前策略。
- 生成评估数据与可视化结果。

### `core/normalizer.py`
- 对观测/目标做归一化，稳定训练。

### `core/util.py`
- 工具函数集合（动作选择、tensor 转换、奖励计算等）。

### `core/logger.py`
- 日志统一封装，便于调试与实验记录。

### `core/HER.py`
- HER（Hindsight Experience Replay）相关采样实现。
- 从 README 看当前主流程中“可能未启用或非主路径”。

## 5. `Env/` 环境模块逐个介绍

## 7. 代码执行关系（简版）

1. 在 `arguments.py` 设置参数。  
2. 运行 `train.py` 启动多进程训练。  
3. `actor.py` 采样 → `buffer.py` 存储 → `learner.py` 更新模型。  
4. `evaluator.py` 周期评估并输出结果图。  
5. 使用 `plot.py` 汇总绘图，或用 `rollout_test.py` 回放策略。
